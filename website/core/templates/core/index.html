About the Game

Welcome to the 'Schlangenprogrammiernacht', a programming game initially developed for GPN19. This is the „official“ server for the GPN20.

Program your own snake’s behaviour, eat as much as you can and try not to die. Remember: always keep a healthy distance to other snakes!

Everybody is invited to join! Simply create an account and start coding.

If you never played SPN before, you may want to watch the Introduction Talk at GPN18 and the presentation of version 2 at GPN19 (both in German).

About this Server

This game is running on a Raspberry Pi 4 with 4 GB RAM. The CPU clock is fixed to the maximum frequency (1.5 GHz) for consistent timing.

Head over to the Grafana Dashboard for gameserver statistics.

What’s new?

Most important new feature: the game can now be played in different programming languages! So far, we have frameworks for C++ and Rust!

Otherwise, we are using the same parameters as on the rC3 2021:

  • Very few „static“ food items.
  • Food decays very slowly.
  • Boosting is very effective (3x speed), but also quite expensive (3 ‰ mass per frame).

Enjoy!
